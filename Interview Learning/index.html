Interview Questions

1. Debouncing:

Debouncing is nothing more just limiting the rate with which a function is executed or called this function is generally going to be a expensive function.

2. Throttling:

Throttling is also just calling a expensive function and limiting the rate of execution of that function. It is just calling the function within some time intervals in between.

It can be tested by moving cursor inside the Throttling box will provide the x and y co-ordincates
