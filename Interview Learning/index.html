9. Pseudo Classes:

Pseudo-classes are keywords that can be added to selectors in CSS to target elements based on their state or position in the document tree. They allow you to style elements in dynamic ways that go beyond simply selecting elements based on their tag names, IDs, or classes.
  • :active - Selects an element while it is being activated by the user.
  • :checked - Selects input elements that are checked (e.g., checkboxes, radio buttons).
  • :default - Selects the default among a group of options.
  • :disabled - Selects input elements that are disabled.
  • :empty - Selects elements that have no children.
  • :enabled - Selects input elements that are enabled.
  • :first - Selects the first matched element of a selector.
  • :first-child - Selects an element if it is the first child of its parent.
  • :first-of-type - Selects the first element of its type within its parent element.
  • :focus - Selects an element when it gains focus.
  • :hover - Selects an element when the user hovers over it.
  • :indeterminate - Selects input elements that are in an indeterminate state (e.g., checkboxes).
  • :in-range - Selects input elements with a value within a specified range.
  • :invalid - Selects input elements with an invalid value.
  • :lang() - Selects elements based on the value of their lang attribute.
  • :last-child - Selects an element if it is the last child of its parent.
  • :last-of-type - Selects the last element of its type within its parent element.
  • :link - Selects links that have not yet been visited.
  • :not() - Selects elements that do not match a given selector.
  • :nth-child() - Selects an element if it is the nth child of its parent.
  • :nth-last-child() - Selects an element if it is the nth child of its parent, counting from the end.
  • :nth-last-of-type() - Selects an element if it is the nth child of its parent of a specific type, counting from the end.
  • :nth-of-type() - Selects an element if it is the nth child of its parent of a specific type.
  • :only-child - Selects an element if it is the only child of its parent.
  • :only-of-type - Selects an element if it is the only element of its type within its parent element.
  • :optional - Selects input elements that are not required.
  • :out-of-range - Selects input elements with a value outside a specified range.
  • :read-only - Selects input elements that are read-only.
  • :read-write - Selects input elements that are not read-only.
  • :required - Selects input elements that are required.
  • :root - Selects the root element of the document.
  • :target - Selects the target element (the one whose ID matches the fragment identifier of the URL).
  • :valid - Selects input elements with a valid value.
  • :visited - Selects links that have been visited by the user.

10. Pseudo Elements:

Pseudo-elements are similar to pseudo-classes, but they allow you to style specific parts of an element rather than the whole element itself. Here's a list of commonly used pseudo-elements in CSS along with their definitions:
  • ::after - Represents a virtual element that is inserted after the content of an element.
  • ::before - Represents a virtual element that is inserted before the content of an element.
  • ::first-letter - Selects the first letter of the first line of a block-level element.
  • ::first-line - Selects the first line of a block-level element.
  • ::selection - Represents a portion of an element that is selected by the user.
  • ::placeholder - Represents placeholder text in an input field or textarea element.
  • ::marker - Represents the list item marker of an element.
  • ::backdrop - Represents the background of the entire page, behind any other elements.
  • ::spelling-error - Represents a word that the browser's spell checker has identified as misspelled.
  • ::grammar-error - Represents a grammatical error in a text.
  • ::placeholder-shown - Represents an input element when its placeholder text is visible.
  • ::before-input - Represents the contents of an input element before the user inputs any value.
  • ::after-input - Represents the contents of an input element after the user inputs a value.
  • ::cue - Represents the marker of a cue in a media element (e.g., a video or audio player).
  • ::file-selector-button - Represents the button used to open the file selector dialog in an input element with type="file".
  • ::outside - Represents the area outside an element's border.
  • ::inside - Represents the area inside an element's border.
  • ::content - Represents the content of an element generated using the content property in CSS.
  • ::region - Represents a container that may be generated by a CSS Regions module.
  • ::part() - Represents a part of a shadow host element that has been defined using the part attribute.
